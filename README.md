## 📁 Project Structure

```
ramadan-coding/
│
└── src/
    │
    ├── assets/                     # Multi asset                
    │
    ├── components/                 # Reusable HTML components
    │   ├── navbar.html             # Global navigation bar
    │   ├── footer.html             # Global footer
    │   └── loader.html             # Page loader
    │
    ├── js/                         # JavaScript per halaman
    │   └── *.js
    │
    ├── pages/                      # Halaman HTML (multipage)
    │   └── *.html
    │
    ├── styles/                     # CSS per halaman
    │   └── *.css
    │
    ├── favicon.ico                 # Favicon
    ├── global.css                  # Global styles
    ├── main.js                     # Main JavaScript
    └── index.html                  # Entry point
```

## 💻 Menjalankan Secara Lokal

### 1. Clone Repository

```bash
git clone https://github.com/adrianwirawicaksana/ramadan-coding.git
cd ramadan-coding
```

### 2. Buka di Browser

```bash
# Windows
start src/index.html

# Mac
open src/index.html

# Linux
xdg-open src/index.html
```

> **Catatan:** Project ini murni static (HTML + CSS + JS) — tidak perlu install dependencies atau menjalankan server.
